            # Goland 使用心得

            # Goland 简介

            oland 是由 JetBrains 公司开发的一个新的商业 IDE，旨在为 Go 开发者提供的一个符合人体工程学的新的商业 IDE。Goland 整合了 IntelliJ 平台（一个用于 java 语言开发的集成环境，也可用于其他开发语言），提供了针对Go语言的编码辅助和工具集成。

            # Goland 下载和安装

            官网下载地址：https://www.jetbrains.com/go/download

            安装过程略过，下一步即可。

            该编辑器是收费的，自己解决，该教程不做探讨。

            # 安装插件

            1. 进入设置 goland -> Preferemces 或者快捷键 ⌘ + ,

            2. 进入插件管理 Plugins

            3. 举个例子🌰，安装中文插件 Chinese (Simplified) Language Pack / 中文语言包

            检索 Chinese 搜索结果中下载量最高的就是我们要找的中文包，点击 Install下载安装即可，安装后需要重启编辑器，重启完成后生效。

            1. 推荐我常用的一些插件

              插件名称 说明
              Chinese (Simplified) Language Pack / 中文语言包 添加中文作为 IDE 界面语言
              EnvFile 配置文件辅助插件（.env、YAML、JSON）
              GitToolBox 在编辑器中显示每一行的 git 提交记录
              Material Theme UI Lite 好看的 UI 插件
              One Dark theme 好看的 UI 插件
              Translation 非常 nice 的翻译插件

            # Goland 常用的快捷键

            这里说一下Mac 的快捷键，windows用户可以将command修改为ctrl

            1. 文件操作相关的快捷键
            快捷键 作用
            command + E 打开最近浏览过的文件
            Shift + Shift 进入万能搜索
            1. 代码格式化相关的快捷键
            快捷键 作用
            command+ Shift + L 格式化代码
            command+ 空格 代码提示
            command+ / 单行注释
            command+ Shift + F 全文件搜索
            command + “. 或 -” 可以将当前（光标所在位置）的方法进行展开或折叠
            F2 页面内跳转到警告位置
            1. 工具窗口
            快捷键 作用
            command+ 1 展示或隐藏项目文件列表栏
            command+ 2 收藏夹
            command+ 3 git 提交
            command+ 4 终端
            command+ 5 调试
            command+ 6 问题
            command+ 7 结构
            command+ 8 服务
            command+ 9 Git
